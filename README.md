# CodeAlpha_StockProtfolioTracker
A simple text-based Stock Portfolio Tracker built in Python
as part of the CodeAlpha Python Programming Internship.

## 📌 About
The user inputs stock symbols and quantities.
The program calculates total investment value using
hardcoded stock prices and optionally saves the report.

## 🧠 Key Concepts Used
- dictionary — to store stock symbols and prices
- input/output — to take user stock selections
- basic arithmetic — to calculate investment values
- file handling — to save report as .txt or .csv

## 🗂️ Features
- 7 predefined stocks (AAPL, TSLA, GOOGL, AMZN, MSFT, META, NFLX)
- Displays a formatted portfolio summary table
- Calculates total investment value
- Save report as .txt or .csv file (optional)
- Option to track another portfolio after each session

## ▶️ How to Run
Make sure Python is installed, then run:
    python stock_portfolio_tracker.py

## 🛠️ Built With
- Python 3
